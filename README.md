# CI/CD Pipeline Project

## Overview
This project demonstrates a complete CI/CD pipeline using GitHub Actions.

## Features
- Automatic workflow trigger on push and pull requests
- Python dependency installation
- Automated unit testing using pytest
- Application build process
- Deployment simulation for staging
- Smoke testing after deployment
- Pipeline status notifications

## Workflow Steps
1. Checkout repository
2. Setup Python environment
3. Install dependencies
4. Run tests
5. Build application
6. Deploy application
7. Run smoke tests
8. Display pipeline status

## Technologies Used
- Python
- Pytest
- GitHub Actions

## How Pipeline Works
Whenever code is pushed to the main branch or a pull request is created, GitHub Actions automatically executes the CI/CD workflow.
